##  English

This academic project presents a **theoretical analysis of the 8 Queens Puzzle**, a classic problem in computer science and constraint satisfaction problems (CSP).

The goal of the puzzle is to place **eight queens on a standard 88 chessboard** so that none of them attack each other, meaning no two queens share the same row, column, or diagonal.

This work focuses on:
- Explaining the problem from an algorithmic perspective
- Analyzing its computational complexity
- Comparing different solution strategies
- Highlighting the importance of data structures in performance optimization

锔 **Note:** This is a **purely theoretical project** and **does not include any code implementation**.

###  Topics Covered
- History of the 8 Queens Puzzle
- Constraint Satisfaction Problems (CSP)
- Backtracking algorithm
- Data structures for optimization
- Comparison between naive and optimized approaches
